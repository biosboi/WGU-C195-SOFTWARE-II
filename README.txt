Database Management System GUI Program

The purpose of this application is to provide a GUI interface for viewing and interacting with the company database.
The program provides the ability to access customer data (add, modify, and delete), as well as appointment data.
Reports are provided in the Reports section to view a summary of all Customer, Contact, and Country data.
